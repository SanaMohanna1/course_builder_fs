# הסבר מקצועי על מערכת Course Builder

## הסבר בפסקה אחת

**Course Builder** היא מיקרו-שירות מבוסס AI שמארגן יצירה, העשרה וניהול אוטומטיים של תוכניות למידה מודולריות. המערכת ממירה נתיבי למידה מותאמים אישית או תוכן שמספקים מדריכים לקורסים מובנים ומוכנים לפרסום, תוך יצירת מבנה היררכי של נושאים, מודולים ושיעורים. המערכת משתמשת ב-Google Gemini AI להעשרת תוכן אוטומטית עם מטא-דאטה, תגיות ומשאבים חיצוניים (YouTube, GitHub), ומספקת מערכת ניתוב חכמה המזהה באופן אוטומטי את סוג הבקשה ומנתבת אותה למטפלים המתאימים תוך יצירת שאילתות SQL דינמיות. המערכת מובנית כארכיטקטורה מודרנית עם Backend ב-Node.js ו-Express (מועלה ב-Railway), Frontend ב-React 19 עם Vite ו-Tailwind CSS (מועלה ב-Vercel), ובסיס נתונים PostgreSQL ב-Supabase. המערכת משלבת 8+ מיקרו-שירותים כולל Learner AI, Content Studio, Assessment, Directory, Learning Analytics, HR, DevLab ו-Skills, תוך שימוש ב-REST ו-gRPC לתקשורת בין שירותים, עם אבטחה מתקדמת המבוססת על OAuth2, RBAC, הצפנת AES-256 ו-TLS 1.3, ותואמת GDPR. המערכת תומכת במחזור חיים מלא של למידה החל מיצירת קורסים, פרסום במרקט-פלייס פנימי, הרשמה, מעקב התקדמות, הערכות וקבלת משוב, תוך הפצת נתונים אנליטיים למיקרו-שירותים משותפים לניהול וניתוח נתוני למידה.
